PacketLogger Fabric 1.20.1

1. Requirements
 - Java 17 (JDK 17)
 - Gradle 8.x (if you want to build locally without wrapper)

2. Build locally
 - In this folder, run:
     gradle wrapper --gradle-version 8.7
     ./gradlew build  (Linux/macOS)
     gradlew.bat build  (Windows)

3. Output
 - Built jars will be in build/libs

4. Minecraft
 - Put the jar in your .minecraft/mods folder for Fabric 1.20.1 with Fabric API installed.
